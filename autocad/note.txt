新建： ctrl + n
打开： ctrl + o
保存： ctrl + s

缩放： 鼠标滚轮向上（放大） 向下(缩小)
平移： 按住鼠标滚轮移动

缩小到纸张大小：z + 空格 + a + 空格
放大到最大视图：双击鼠标滚轮

图形选择：点选(鼠标移动到选择的对象上单击左键)
          左选(点击图纸上一点按住鼠标左键，从左到右完全框住所选对象)
          右选(点击图纸上一点按住鼠标左键，从右到左关联到所选对象即可)
          选择是一种状态，具有连续性，按esc退出选择状态

对象删除：按照图形选择选择要删除的对象，按e + 空格/图标/右键进行删除

图片导入：选择图片 ctrl + c, 然在图纸上选择插入点 ctrl + v, 然后调整旋转后即可

对象捕捉：F3(打开关闭捕捉模式)，点击最下面的菜单，找到捕捉设置，设置要捕捉的端点（端点/圆心/中点/切点等等）
